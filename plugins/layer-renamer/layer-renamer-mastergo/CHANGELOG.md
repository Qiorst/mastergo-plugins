# Changelog

## 1.0.0 - 2026-06-29

- Initial MasterGo release.
- Added batch renaming for selected layers.
- Added block-based template builder with removable, draggable, and editable text blocks.
- Added `{date}`, `{n}`, `{i}`, `{old}`, and `{type}` template tokens.
- Added date picker and date format selector.
- Added live preview, selection diagnostics, empty-selection handling, and rename result reporting.
- Added Chinese default UI with Chinese/English switching.
